# Future Time

This is a very small React app built with Vite. The project has been cleaned so it contains only what is needed to run the app.

## How to run it

1. Install dependencies (only React + Vite tooling):

   ```bash
   cd future-time
   npm install
   ```

2. Start the dev server:

   ```bash
   npm run dev
   ```

3. Open the URL that the terminal shows you (usually `http://localhost:5173`).

## How to build for production

```bash
npm run build
```

The built files will be in the `dist` folder (which is ignored by Git).
